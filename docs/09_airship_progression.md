# 09 - Airship Progression

## Core Rule

Players never receive an airship for free.

All ships are built from scratch using Create Aeronautics systems.

## Airship Progression Phases

1. Prototype Phase
2. Operational Scout Phase
3. Expedition Carrier Phase
4. Flying Base Phase

## Rank-Gated Components

- Large engines
- Critical control systems
- Heavy structural modules
- Long-range logistics modules

## Late-Game Flying Base Requirements

- Create factory segment
- Mekanism factory segment
- Storage wing
- Living area
- Workshop area

## Design Objectives

- Airship construction is a major campaign pillar, not a side gimmick.
- Engineering effort should produce visible strategic advantages.
- Larger ships should improve autonomy for multi-region expeditions.

## Progression Constraints

- No bypassing rank gates via market exploits.
- Upgrades should require mixed inputs: quests, materials, and milestone clears.
- Ship loss or damage events should create stories without hard-reset frustration.
