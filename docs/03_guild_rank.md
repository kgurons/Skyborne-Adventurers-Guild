# 03 - Guild Rank Progression

## Rank Ladder

- D
- C
- B
- A
- S
- EX

## Shared Progression Principles

- Rank is shared across all Guild members.
- Rank never decreases.
- Progress should require effort but avoid grind walls.
- Advancement should combine multiple activity types.

## Unlock Categories per Rank

- NPC access and new dialogue paths
- Shop inventory and services
- Airship engines and structural components
- Main story chapters
- Dungeon tiers
- Boss encounters
- Region access and expedition permits

## Example Advancement Requirements

Each rank-up should mix at least four requirement types:

1. Quest completion threshold
2. Dungeon clear milestone
3. Boss hunt objective
4. HQ upgrade or construction objective
5. Exploration discovery count
6. Emergency response completions

## Rank-Up Ceremony

- Trigger a Guild Meeting event after each rank up.
- Add NPC reactions and ceremonial rewards.
- Unlock next strategic planning board at HQ.

## Balance Targets

- D to C: onboarding, low friction
- C to B: first coordinated expedition arc
- B to A: midgame specialization and stronger bosses
- A to S: advanced airship and multi-phase encounters
- S to EX: legacy tier challenges and finale arcs
